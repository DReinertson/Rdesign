The Table

These beautiful upcycled tables are made with reverseable pressed wood, lightly sanded and finished to give a rich walnut / cognac color on the surface.

The routed X design has been painted black on one side and gunmetal on the other, with the metal base painted to compliment the top.

Table Specs

  • Reverseable pressed wood top
  • Cognac brown, shade varies slightly with each table
  • Black metal base
  • Stackable to create shelving
  • Stores flat
  • Felt floor protector pads on base
  • Dimensions 30"L x 30"W x 18 3/4"H
  • Available in pink, green, classic blue, black & orange
