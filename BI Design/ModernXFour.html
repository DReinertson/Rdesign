The Modern X Table

Beautifully upcycled wood and metal table that fits an industrial glam lifestyle. May be used as a coffee, side, or cocktail table. Can stack multiple tables to create a shelving unit. Great for home or office! Limited edition.

    Dimensions
  • 30"L x 30"W x 18 3/4" H
    Material
  • Pressed wood top, metal base
    Features
  • Reversible Top: The routed X design painted black on top and silver on the reverse, makes the top reversible from a black X to a silver X
  • Stackable: The routed X design on the table top accommodates the base of the next table for stackability, stack up to four tables to create shelving
  • Stores Flat: Disassembles to store or ship flat.
  • Customize: A few tables have been set aside for custom color orders, see color ideas below
  • Felt pads on base to protect flooring
    Description
  • Each table has been lovingly upcycled and refurbished with careful attention to detail but there may be inherent, subtle variations in wood tone color from piece to piece as well as minor imperfections in the wood or metal. Pressed wood top has been sanded and refinished, metal base has been repainted. In their former life, these tables were used as store fixtures, meaning they are commercial grade quality and now also possess the finishing touches and beauty you expect for your home.
